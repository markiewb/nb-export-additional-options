nb-export-additional-options
==================================

This plugin adds some more options to export your NetBeans settings.

<p>
Features:
<ul>
<li>Configuration proposed by Adam Bien (SVN, Favorites, Templates/user.properties, projectui, editors/text)</li>
</ul>
</p>
